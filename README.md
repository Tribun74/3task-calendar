# 3task Calendar – Event Calendar for WordPress

A clean, lightweight event calendar plugin for WordPress. Create and manage events with categories, locations, and a beautiful calendar view.

[![WordPress Plugin Version](https://img.shields.io/wordpress/plugin/v/3task-calendar)](https://wordpress.org/plugins/3task-calendar/)
[![WordPress Plugin Downloads](https://img.shields.io/wordpress/plugin/dt/3task-calendar)](https://wordpress.org/plugins/3task-calendar/)
[![License: GPL v2](https://img.shields.io/badge/License-GPL_v2-blue.svg)](https://www.gnu.org/licenses/gpl-2.0)

## Features

- Monthly calendar view with event indicators
- Event categories with color coding
- Location management
- Shortcode support (`[3task_calendar]`)
- Gutenberg block support
- Mini calendar widget
- Email notifications for upcoming events
- Fully responsive + dark mode
- GDPR-friendly (no external services)
- Lightweight and fast

## Installation

1. Upload `3task-calendar` to `/wp-content/plugins/`
2. Activate the plugin in WordPress
3. Go to **3task Calendar** in your admin menu
4. Add your first event

Or install directly from [WordPress.org](https://wordpress.org/plugins/3task-calendar/).
